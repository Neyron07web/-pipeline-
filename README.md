Игра с фильтрами на изображениях (Filter Game)

Описание проекта:

Эта программа реализует обучающую игру на Python с использованием OpenCV и PIL, где пользователю предлагается угадывать фильтры, применённые к изображению. Игра ориентирована на школьников и студентов, помогает развивать внимательность и понимание базовых графических эффектов.

---

Основные возможности

1. Случайный выбор изображений из набора (например, машина, кот, город, собака, дом).  
2. Применение 1–3 фильтров одновременно:
   - Размытие (Blur) — слегка размывает изображение
   - Контуры (Edges) — выделяет границы объектов
   - Чёрно-белый (Threshold) — упрощает картинку в ч/б
   - Инверсия (Invert) — меняет цвета на противоположные
   - Резкость (Sharpen) — повышает чёткость деталей
   - Тиснение (Emboss) — добавляет рельеф
   - Мультяшный (Cartoon) — упрощает цвета и выделяет контуры
3. Онлайн-панель с описанием фильтров, количеством фильтров на картинке, таймером и баллами.  
4. Система угадывания:
   - Правильный выбор фильтра: сообщение "Правильно! Осталось найти X фильтр(ов)".  
   - Неправильный выбор: сообщение "Неправильно! Следующая картинка" + пауза 5 секунд.  
5. Таймер на каждый уровень — 30 секунд на угадывание фильтров.  
6. Баллы и уровни:  
   - Игра состоит из 5 картинок (уровней).  
   - За полностью угаданную картинку начисляется 1 балл.  
   - После 5 картинок выводится итоговый счёт.

---

Установка и запуск

1. Клонировать репозиторий:  
  git clone <URL_репозитория>
2. Установить зависимости:
   pip install opencv-python pillow numpy
4. Добавить изображенияв папку images/ (по умолчанию используются c63.jpg, cat.jpg, city.jpg, m5f10.jpg, x5m.jpg
5. Запустить игру:
   python filter_game.py
